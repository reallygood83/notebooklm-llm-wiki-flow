# Obsidian integration

This project follows an Obsidian-friendly property strategy inspired by Hermes obsidian skills.

## Property design
- `source_notes`: internal references inside the vault
- `source_urls`: external links that can be clicked directly from Obsidian properties
- `share_link`: optional ShareNote property

## Starter kit
Users without Hermes obsidian skills can install a starter kit into any vault:

`nlwflow install-obsidian-kit --vault /path/to/vault`

This creates:
- `000-Inbox/NLW Getting Started.md`
- `100-Templates/NLW Comparison Template.md`
- `100-Templates/NLW Query Template.md`
- `900-System/nlwflow-obsidian-skills-guide.md`
